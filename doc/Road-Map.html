Here, some of the features will be implemented in next versions:

  • MVC (Multi Page) version for the backend (similar to current SPA Angular version).
  • Logging and viewing success/fail login attempts.
  • Enhancements on front-end (like newsletter and contact form)
  • Basic content management system.
  • Hierarchical Organization Unit (groups) system.
  • Edition/Package system for multi-tenant applications.
  • Persistent Notification system.
  • Ordering menu items.
  • Simple background job queue.
  • Sending emails to administrators on errors.
  • Improvements on current features.
  • ASP.NET 5 (vNext) version when it's released.
  • Angular 2.0 version after it's released.

All these features and much more will be available also for current customers in their upgrade period.
